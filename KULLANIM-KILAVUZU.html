WhatsApp Isletme AI Kullanım Kılavuzu

Bu paket temiz satış sürümüdür. Hiçbir kişisel API anahtarı, WhatsApp oturumu veya özel veri içermez.

1. Bu paket ne yapar?

Bu yazılım, WhatsApp konuşmalarını yapay zeka ile cevaplayan, müşterileri operatöre devreden, şirket bilgisi ve belge tabanından yanıt üreten yerel bir panel uygulamasıdır. Tüm veriler bilgisayarınızda saklanır.

AI anahtarınızı dosya içine yazmanız gerekmez. Panel açıldıktan sonra kendi OpenAI, Gemini, Claude, DeepSeek veya OpenRouter hesabınızı girersiniz.

2. Gerekli olanlar

Windows

Windows 10 veya Windows 11.

Node.js

Kurulum için gerekli. Resmi indirme sayfası: nodejs.org/en/download

WhatsApp hesabı

Panelden QR kod ile giriş yapılır. Aktif bir WhatsApp hesabı gerekir.

İnternet

AI servisleri, model listesi ve WhatsApp bağlantısı için internet gerekir.

Önerilen ücretsiz AI

En kolay başlangıç için Google AI Studio üzerinden Gemini API key alın. API key sayfasını aç

Chrome veya Edge yüklü olması önerilir. Uygulama tarayıcı tabanlı oturum açar.
Bilgi tabanındaki anlamsal arama ilk açılışta ek model indirebilir. İnternet yoksa sistem otomatik olarak hafif bir anahtar-kelime aramasına düşer; çalışma durmaz.

3. Dosya yapısı

  • kurulum.bat: Bağımlılıkları kurar ve projeyi derler.
  • Baslat.bat: Uygulamayı açar ve paneli başlatır.
  • scripts/start-panel.ps1: Teknik başlatma akışı.
  • src/: Uygulamanın kaynak kodu.
  • tests/: Otomatik testler.
  • .env.example: Opsiyonel yerel ayar örneği. İsterseniz bunu .env olarak kopyalayabilirsiniz.

4. Kurulum

  1. Dosyaları bilgisayarınıza çıkarın.
  2. Node.js yüklü değilse resmi sayfadan kurun.
  3. kurulum.bat dosyasına çift tıklayın.
  4. Kurulum tamamlanınca Baslat.bat dosyasını çalıştırın.
Node.js resmi indirme: https://nodejs.org/en/download

5. İlk kullanım

  1. Tarayıcı açılır ve yerel panel yüklenir.
  2. WhatsApp hesabı ekleyin veya varsayılan hesabı başlatın.
  3. QR kodu telefonunuzdan okutun.
  4. AI sağlayıcınızı seçin.
  5. Kendi API anahtarınızı girin ve model listesini alın.
  6. Şirket bilgisi, fiyat, hizmet, kargo ve sık sorulan soruları ekleyin.
  7. PDF, DOCX, TXT veya MD dosyalarını bilgi tabanına yükleyin.

6. En kolay ücretsiz AI ayarı: Google Gemini

Başlangıç için en pratik seçenek Google AI Studio üzerinden Gemini API key almaktır. Ücretsiz kullanım kotaları ve erişilebilen modeller zamanla değişebilir; yoğun veya ticari kullanımda Google tarafında ücretli plana geçmeniz gerekebilir.

Panelde önerilen model: Gemini 2.5 Flash. Daha hafif ve hızlı kullanım için Gemini 2.5 Flash-Lite veya listede görünen benzer Flash modelleri de seçilebilir.

Gemini API key alma adımları

  1. Google hesabınıza giriş yapın.
  2. Google AI Studio API key sayfasını açın: https://aistudio.google.com/app/apikey
  3. İlk girişte şartları kabul edin.
  4. Create API key düğmesine tıklayın.
  5. Yeni proje oluştur seçeneğini kullanın veya mevcut Google Cloud projenizi seçin.
  6. Oluşan API key değerini kopyalayın ve güvenli yerde saklayın.

Resmi bağlantılar

  • Google AI Studio API key sayfası
  • Gemini API key resmi dokümanı
  • Gemini model listesi
  • Gemini ücretsiz/ücretli kullanım ve fiyatlandırma

Bu programda Gemini ayarı

  1. Baslat.bat ile paneli açın.
  2. AI Sağlayıcısı bölümünde Google Gemini seçin.
  3. API anahtarı alanına Google AI Studio'dan aldığınız key değerini yapıştırın.
  4. Modelleri getir düğmesine tıklayın.
  5. Model listesinden Gemini 2.5 Flash, Gemini 2.5 Flash-Lite veya listede görünen benzer Flash modelini seçin.
  6. AI ayarlarını kaydet düğmesine tıklayın.
API key değerini kimseyle paylaşmayın, ilan açıklamasına veya herkese açık dosyaya koymayın. Müşteri sadece kendi paneline girer.

7. Panelde doldurulacak alanlar

  • AI sağlayıcı seçimi
  • AI model seçimi
  • Kendi API anahtarınız
  • WhatsApp hesap adı
  • Şirket bilgileri ve belgeler
  • İsteğe bağlı .env ayarları: port ve veri klasörü

Bu paket içinde hazır anahtar, şifre veya kullanıcı verisi yoktur. .env.example dosyasındaki satırlar yorumlu durumdadır; sadece ihtiyaç duyduğunuz satırı açın.

8. Veriler nerede tutulur?

Konuşmalar, ayarlar ve bilgi tabanı yerel SQLite veritabanında saklanır. WhatsApp oturumu ve tarayıcı verileri de bilgisayarınızda oluşan yerel klasörlerde tutulur.

Varsayılan konumlar uygulama tarafından otomatik oluşturulur. İsterseniz sonradan ayarlardan değiştirebilirsiniz.

9. Sorun olursa

  • Node.js bulunamadı: Resmi siteden kurun ve tekrar kurulum.bat çalıştırın.
  • Panel açılmıyor: Güvenlik duvarı veya başka bir uygulama aynı portu kullanıyor olabilir.
  • QR gelmiyor: Hesabı yeniden başlatın ve internet bağlantısını kontrol edin.
  • Model listesi gelmiyor: API anahtarını ve sağlayıcı hesabınızı kontrol edin.
  • Gemini kota hatası: Ücretsiz kota dolmuş olabilir. Daha sonra tekrar deneyin, Flash-Lite seçin veya Google tarafında ücretli plana geçin.

10. Kısa kullanım özeti

  1. Node.js kur.
  2. kurulum.bat çalıştır.
  3. Baslat.bat çalıştır.
  4. WhatsApp QR okut.
  5. Google AI Studio'dan Gemini API key al.
  6. Panelde Google Gemini seçip Gemini 2.5 Flash veya benzer Flash modelini kaydet.
  7. Şirket bilgilerini ekle.
  8. Kullanmaya başla.

Bu kılavuz satış paketine özeldir ve müşteri tarafından doğrudan açılıp okunabilir.
